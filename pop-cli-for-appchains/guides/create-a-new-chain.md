# Create a new chain

Use the interactive prompt to scaffold a chain:

```bash
pop new chain
```

Available templates:

- Pop
  - Standard
  - Assets
  - Contracts
- OpenZeppelin
  - Generic Runtime Template
  - EVM Template
- Parity
  - Polkadot SDK's Parachain Template
