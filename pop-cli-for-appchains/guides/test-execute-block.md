# Test execute-block

Use `pop test execute-block` to execute a block with `try-runtime` in a controlled environment.

## Prerequisites

- A runtime built with the `--try-runtime` feature.
- Pop CLI installed with the `chain` feature.

## Usage

```bash
pop test execute-block [OPTIONS]
```

## Example

```bash
pop test execute-block --help
```

## Notes

Use `--help` to see the required inputs and available flags for your runtime and target chain.
