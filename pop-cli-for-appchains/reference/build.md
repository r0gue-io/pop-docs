# build

```bash
pop build <COMMAND>
```

To build your parachain using Pop CLI:

```bash
# Build your parachain
pop build parachain -p ./my-appchain
```

or

```bash
cd my-app
pop build parachain
```

To build your parachain for production:

```bash
cd my-app
pop build parachain --release
```
